Etiquetas Básicas de HTML

HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web está "marcado" con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto. Estas etiquetas de marcado son las propias etiquetas HTML.

El hipertexto (HTML) es un lenguaje informático que forma la mayoría de las páginas web y plataformas online. HTML no se considera un lenguaje de programación, ya que no puede crear una funcionalidad dinámica. Sin embargo, los usuarios web pueden crear y estructurar secciones, párrafos y enlaces usando elementos, etiquetas y atributos.

Sin embargo, el HTML ha ido evolucionando los últimos años. Para hacerse a una idea, la primera versión contaba solamente con 18 etiquetas. Desde entonces, cada nueva versión ha traído nuevas etiquetas y atributos. La actualización más importante fue la introducción en 2014 del HTML5. La principal diferencia entre ambas es que la nueva admitía nuevos tipos de controles de formularios. También incluía diversas etiquetas semánticas que describían mejor el contenido, como "article", "headers" y "footer".

Atributos y Valores

Hay una serie de etiquetas que son las más usadas para crear cualquier documento HTML, como las siguientes:

<body>: para el contenido.

<body> </body>

Indica la parte del cuerpo del contenido de un documento HTML. Es una etiqueta esencial para cualquier documento ya que indica donde empieza el contenido visible del documento.

<head>: para información sobre el documento

<head> </head>

La parte superior del documento HTML, es donde se indican los metadatos: título del documento, hojas de estilos, javaScript, CSS…

<div>: división dentro del contenido

<div> </div>

Un elemento que es usado mayoritariamente para agrupar otros elementos y actuar como plantilla de otros controles. La etiqueta ayuda a estructurar el documento en secciones.

<a>: para enlaces

<a> </a>

Es una etiqueta que ayuda a crear un enlace a una página web. El atributo principal de la etiqueta HTML es href, donde se pone el enlace al que se quiere conectar. Otro atributo muy usado es target, el cual sirve para indicar si el enlace se abrirá en una nueva ventana o en la misma.
Ejemplo HTML:
Pulsa <a href=”https://www.nombredelaweb.com/” target=”_blank”>aquí/a> para visitar DonDominio.

<stron>: para poner el texto en negrita

<stron> </stron>

Con la etiqueta strong se resalta texto o se pone en "negrilla".

<br>: para saltos de línea

<br> </br>

Con esta etiqueta HTML le indica al navegador que viene un salto de línea. Sirve para hacer el texto más leíble.

<h1...h6>: para títulos dentro del contenido

<h1> </h1>... <h6> </h6>

Hay diferentes niveles de títulos, del 1 al 6. Las etiquetas <H + número> permiten indicar la importancia del título y para estructurar el contenido.

<img>: para añadir imágenes al documento

<img> </img>

Se usa la etiqueta IMG para mostrar imágenes dentro del contenido. Necesita el atributo src para funcionar, ya que será donde se indica desde donde tiene que mostrar la imagen.

<ol>: para listas ordenadas, <ul>: para listas desordenadas, <li>: para elementos dentro de la lista

<ol> <li> </li> <ol> | <ul> <li> </li> <ul>

Las etiquetas OL y LI sirven para crear listas, OL para listas ordenadas y UL para listas sin orden. Dentro de las listas, los elementos se identifican con la etiqueta LI.
Ejemplo HTML:
<ul>
<li>Primer elemento</li>
<li>Segundo elemento</li>
<ul>

<p>: para parágrafos

<p> </p>

Etiqueta que sirve para agrupar texto dentro de un parágrafo. El propósito es poder hacer el contenido más fácil de leer y organizado.

<span>: para estilos de una parte del texto

<span> </span>

Con la etiqueta se personaliza el estilo de solamente una parte del texto.
Ejemplo HTML:
Solo <span style=”color: blue;”>esta palabra/span> en azul.
